# Numerical Methods

A C++/OpenCL library for numerical methods.

## Currently implemented methods: 

### Integrators for ordinary differential equations (ODEs)

- explicit Runge-Kutta methods
	- euler
	- midpoint
	- rk4
	- fehlberg54
	- dopri54
	- cashkarp54
	- bosha32
- tayler series expansions for exponential functions

## Depdendencies

- noma_bmt
- noma_ocl
- noma_typa
